# Shared configs

Reserved for claim-neutral config fragments that are reused by multiple EQ-Fuzzy workstreams.

Do not add paper-specific experiment YAML here. A config belongs here only after at least two workstreams need the same registry or execution setting.

Current shared configs include the model panels and source-derived text/persona registries used by ICECCME and expected to be reused by later workstreams.

The shared private text bodies are local, untracked inputs under
`data/catalogs/texts_private/`. The tracked registries here describe those text
ids and metadata; they do not contain the text bodies.
