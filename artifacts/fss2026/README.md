# FSS 2026 artifacts

Generated artifacts for the FSS 2026 membership-specification and
sensitivity-reporting workstream.

The initial artifact set is expected under:

- `membership_specification_v1/`

Expected outputs:

- `membership_grid.csv`
- `entropy_response_grid.csv`
- `membership_specification_card.csv`
- `membership_specification_card.tex`
- `family_sensitivity_matrix.csv`
- `family_sensitivity_matrix.tex`
- `figure1_membership_and_entropy_response.png`
- `figure_summary.json`
- `analysis_summary.json`

These artifacts are score-axis artifacts only. They must not include model
names, provider names, persona labels, temperature levels, SCIS factorial
decomposition values, SCIS heatmaps, ICECCME human-alignment rankings, ICICIC
benchmark-positioning tables, target-shift tables, or raw JSONL files.
