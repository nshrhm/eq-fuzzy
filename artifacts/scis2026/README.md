# SCIS 2026 artifacts

Generated artifacts for the SCIS 2026 workstream.

Current artifact sets:

- `pilot_analysis_v1/`: Phase 5.5 pilot analysis-readiness outputs.
- `main_analysis_v1/`: emotion-level long tables, entropy tables, cell
  summaries, and factorial decompositions for the repaired main run.
- `main_inspection_v1/`: descriptive main-run inspection outputs used to select
  paper tables and figures.
- `main_tables_v1/`: manuscript-facing CSV and LaTeX table artifacts.
- `main_figures_v1/`: manuscript-facing PNG figures, LaTeX include snippets,
  and the figure manifest.
- `main_bootstrap_v1/`: within-cell bootstrap confidence intervals for the
  primary decomposition table.
- `main_reviewer_diagnostics_v1/`: retry, leave-one-model-out,
  story-stratified, and emotion-stratified reviewer-risk diagnostics.

These artifacts are SCIS-owned and should not be mixed with ICECCME or ICICIC
outputs.

The main-run reproducibility map is recorded in
`docs/scis2026/phase16_reproducibility_package.md`.
