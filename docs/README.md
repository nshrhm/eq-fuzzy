# Documentation ownership

Repository-level policy and cross-workstream navigation live directly under `docs/`.

Workstream-specific notes live under the matching workstream directory:

- `docs/iceccme2026/` for ICECCME 2026 run guides, output inventories, reproducibility notes, and paper-planning notes.
- `docs/scis2026/` for SCIS 2026 planning notes. Keep this minimal until the real experiment design is fixed.
- `docs/icicic2026/` for ICICIC 2026 planning notes. Keep this minimal until the real experiment design is fixed.

The canonical reusable context prompts remain in `docs/context/` because they are used to orient future work across the monorepo.

Do not place paper-specific claims, run guides, or generated-output inventories directly under `docs/`.
