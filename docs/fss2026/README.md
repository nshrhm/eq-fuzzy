# FSS 2026 docs

This directory is reserved for FSS 2026 planning and operational notes.

FSS 2026 focuses on membership-function specification and sensitivity
reporting for fuzzy-entropy-based LLM emotion scoring. Its main artifacts are
score-axis membership grids, entropy-response curves, a membership
specification card, and a family-pair sensitivity matrix.
